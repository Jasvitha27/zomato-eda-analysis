# 📊 Zomato Restaurant Data Analysis

Exploratory Data Analysis on Zomato dataset to understand factors affecting restaurant ratings using Python, Pandas, and Seaborn.

## 📊 Key Insights

* Votes have a moderate relationship with ratings
* Cost has a weak correlation with ratings
* Location plays an important role in restaurant performance
* Online ordering has minimal impact on ratings

## 🛠️ Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib

## 📁 Dataset

The dataset used in this project can be downloaded from Kaggle:

Zomato Bangalore Restaurants Dataset
https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants
